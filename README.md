# Installation
### Download and change folder
```
cd ~ && \
git clone https://github.com/vdarkobar/dotfiles.git && \
cd dotfiles && \
chmod +x install.sh && \
./install.sh
```
  
# Uninstallation

### Change to dotfiles folder
```
cd ~/dotfiles
```

### Run uninstaller
```
./uninstall.sh
```
